Laplace加罚项

  • 选择增加 \((\Delta u_h, \Delta v_h)_T\) 这样的加罚项

  • 根据 \(R_3\) 的表达，这个加罚项只需要 \(\sigma h^2\) 的因子就可以控制住。

  • 该加罚项的相容性：必须额外利用方程以及右端项 \(f\)，即 \begin{align*} \sigma h^2( \Delta (u - I_h u), \Delta v_h)_{T} \lesssim \sigma h^2 h^{k-1}\| u\|_{H^{k+1}} h^{-1}\|\nabla v_h\|_{T} = \sigma h^{k}\| u\|_{H^{k+1}} \|\nabla v_h\|_{T} \end{align*}

Euler-Lagrage Remapping cut-FEM

用Ritz投影来处理压力L2的不稳定性

这种全欧拉时间步进方案最初在 [33] 中针对对流-扩散问题提出并进行了分析，随后在 [34] 中应用于两相流问题，在 [35] 中用于处理带有浸没移动物体的流体问题。

类似的非拟合方法采用时空单元处理流体问题，见 [36]。

针对变形域中的 Stokes 和线性化 Navier-Stokes 方程的扩展研究分别在 [37–39] 中进行了分析，使用非拟合 Taylor-Hood 有限元方法（任意阶）证明了速度的能量范数和压力的缩放 (L2(H1)) 范数的最优收敛性。

尽管这一方法有坚实的数学基础，但本文发现现有方法的变体在水动力力上的时间伪振荡问题仍未完全解决。

这些伪振荡虽然随着网格加密快速减小，但即便在体积和质量守恒严格满足的情况下仍然存在——这与常见假设（即守恒原理的失效是主要原因）相悖。

此外，与早期研究一致，我们观察到，当时间步长减小时，伪振荡的幅度会增加，尽管时间平均振幅保持有界。这一行为表明，该方法在不同范数下的数值稳定性界限可能有所不同。我们认为，控制导致浸没边界上的伪水动力力的压力伪振荡，需要一种在 (L:nbsphinx-math:infty`(H`1))-范数下对压力无条件稳定的离散化方法。

而传统有限元分析通常仅评估 (L2(L2))-范数下的压力稳定性，这仅保证了压力的时间平均稳定性。在合理假设下，(L:nbsphinx-math:infty`(H`1))-压力稳定性可以归结为 (L:nbsphinx-math:infty`(L`2))-压力稳定性，两者均衡指瞬时压力的稳定性。我们的数值实验表明，目前所有可用的非拟合有限元方法变体在 (L2(L2))-范数下均表现出无条件稳定性，但当时间步长趋近于零且空间网格固定时，压力在 (L:nbsphinx-math:infty`(L`2))-范数下发散。

我们重新审视了稳态域上的有限元方法分析，以识别实现 (L:nbsphinx-math:infty`(L`2))-压力稳定性的关键推论，并解释这些推论为何在移动边界情况下失效。通过明确问题所在，我们提出了一种方法变体，能够确保瞬时压力的无条件稳定性，从而实际上抑制了压力伪振荡，并消除了对特定 CFL 数的依赖。方法的关键修改在于将 ghost-penalty 项重新定义为全局形式，而非此前建议的仅限于边界周围的窄带区域。

除此之外，本文还研究了流体粘性、几何近似、质量守恒、离散化与稳定化参数以及有限元空间选择对窄带浸没边界有限元方法中压力时间伪振荡的发生及幅度的影响。结果表明，其中一些参数对压力的时间稳定性几乎没有影响，而另一些参数的变化则可以将“传统”变体中的伪振荡降低至低于物理显著性的水平。

除了上述发现外，本文还研究了流体粘性、几何近似、质量守恒、离散化与稳定化参数以及有限元空间的选择对窄带浸没边界有限元方法中伪压力时间振荡的发生及幅度的影响。结果表明，其中一些参数对压力时间稳定性几乎没有影响，而另一些参数的调整可以将“传统”方法中的伪振荡降低至低于物理显著性的水平。

3.4. 几何近似

将几何与计算网格分离会带来准确近似几何的挑战，这需要在被截断的单元上使用鲁棒的数值积分技术。文献中提出了多种方法，例如 [40, 48–52]。

在 cutFEM 方法中，标准做法是使用水平集函数的分段线性插值。在单纯形上，这种方法通过将被截断的单元划分为子单纯形并应用标准高斯求积规则实现 [25]。然而，这种分段线性近似会引入几何误差： [ \text{dist}`(:nbsphinx-math:Omega`, \Omega_h) :nbsphinx-math:`lesssim `h^2. ] 对于使用 Taylor–Hood 元的细网格，当 ( k = 2 ) 时，这种几何误差主导 ( L^2 ) 速度误差；对于更高阶元，它会影响 ( L^2 ) 和 ( H^1 ) 速度误差以及 ( L^2 ) 压力误差。

等参非拟合有限元方法 [40]（在移动域场景中已分析 [41]）将几何误差降低到 ( O(h^{k+1}) )。该方法通过计算一个小的网格变形 (\Theta_h)，将分段线性水平集映射到精确水平集的高阶近似。在移动域场景中，关键挑战在于将相对于 (\Theta`^{n-1}) 计算出的速度 (u^{n-1}) 投影到变形 (:nbsphinx-math:Theta`^n) 上，以保持最优收敛性 [41]。

另一种更简单的方法是细分被截断单元用于积分，并在这些细分单元上应用分段线性水平集近似。虽然这种方法可以减少几何误差，但随着网格精度提高，需要更多细分以保持最优收敛性，这在计算上变得不可行。然而，它对于隔离纯几何近似误差（区别于等参方法所需的投影）仍有一定的应用价值。
